# ByteCart

A simple e-commerce platform for tech accessories.

*This project was built as part of the full-stack internship at TechieHelp: https://www.techiehelp.in/*

## Quick Start

```bash
npm install
npm run dev
```

## Features

- Product catalog with search and filtering
- Shopping cart with real-time updates
- Multiple payment options (UPI, COD)
- Order tracking and management
- Responsive design for mobile and desktop
- User authentication and profiles
- Product reviews and ratings
- WhatsApp and SMS order notifications

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- Framer Motion
- React Router
